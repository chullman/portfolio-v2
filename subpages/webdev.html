Application:

"Bubble Buddies"

Likened to the popular group event hosting and organisation platform, "meetup.com", this app was developed from the ground up as a demo/student project to showcase my skills and experience in Rails development. Key phases of the Software Development Lifecycle were conducted and recorded/documented as part of this project.

Source:

https://github.com/chullman/BubbleBuddies

Live Demo: https://bubble-buddies.herokuapp.com

Key Technology:

I had built this application using Ruby on Rails v5.2.0 - a server-side web application framework written in Ruby based on the MVC structure.

I had designed and implemented a rudimentary relational database schema to be executed on a PostgreSQL DBMS.

Certain existing publically sourced APIs were utilised such as Google Maps and Stripe.

Application:

"SAP Dashboard"

Designed and purposed with the objective to provide a more "user-friendly" interface for stakeholders to perform data analytics and gather information from select SAP ERP data marts with excellent User Experience (UX) in mind. Although it is in its prototype stage, the foundational code structure is all in place, ready to be built upon with relative ease with more dashboard functionality.
